# @boa/base

@boa/base package is the core package of boa. It has [AppProvider](#AppProvider), ComponentBase, ComponentComposer, EditorBase, Enums, ErrorBoundary, BOA Themes, and utilities.

## Installation


```sh
// with npm
npm install @boa/base

// with yarn
yarn add @boa/base
```
